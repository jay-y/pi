# 技能管理

本文档介绍如何在 Go SDK 中发现、过滤和管理技能。

## 技能发现

SDK 会自动从以下位置发现技能：

- `~/.pi/agent/skills/` - 用户技能目录
- `<cwd>/.pi/skills/` - 项目技能目录

### 技能发现规则

- **根目录**：直接子目录中的 `.md` 文件
- **子目录**：递归查找 `SKILL.md` 文件

### 技能文件格式

技能文件使用 YAML frontmatter 格式：

```markdown
---
name: my-skill
description: 这是一个示例技能
disable-model-invocation: false
---

# 技能内容

这里是技能的详细说明...
```

## 加载技能

使用 `LoadSkills` 函数从所有配置位置加载技能：

```go
// 加载所有技能
result := session.LoadSkills(session.LoadSkillsOptions{
    Cwd:             "/path/to/project",
    AgentDir:        "~/.pi/agent",
    IncludeDefaults: boolPtr(true), // 包含默认技能目录
})

// 查看加载的技能
fmt.Println("发现的技能:")
for _, skill := range result.Skills {
    fmt.Printf("  - %s: %s\n", skill.Name, skill.Description)
}

// 查看诊断信息
if len(result.Diagnostics) > 0 {
    fmt.Println("警告:")
    for _, diag := range result.Diagnostics {
        fmt.Printf("  [%s] %s: %s\n", diag.Type, diag.Path, diag.Message)
    }
}
```

## 从特定目录加载技能

只从指定目录加载技能：

```go
result := session.LoadSkillsFromDir(session.LoadSkillsFromDirOptions{
    Dir:    "/path/to/skills",
    Source: "custom", // 技能来源标识
})

for _, skill := range result.Skills {
    fmt.Printf("技能: %s\n", skill.Name)
}
```

## 过滤和自定义技能

使用 `DefaultResourceLoader` 过滤或替换技能：

```go
// 定义自定义技能
customSkill := session.SkillInfo{
    Name:        "my-skill",
    Description: "自定义项目指令",
    FilePath:    "/virtual/SKILL.md",
    BaseDir:     "/virtual",
    Source:      "custom",
    DisableModelInvocation: false,
}

// 创建资源加载器，过滤技能
resourceLoader := session.NewDefaultResourceLoader(session.DefaultResourceLoaderOptions{
    SkillsOverride: func(current session.SkillsResult) session.SkillsResult {
        // 过滤只保留包含 "browser" 或 "search" 的技能
        filteredSkills := []session.SkillInfo{}
        for _, skill := range current.Skills {
            if strings.Contains(skill.Name, "browser") || strings.Contains(skill.Name, "search") {
                filteredSkills = append(filteredSkills, skill)
            }
        }

        // 添加自定义技能
        filteredSkills = append(filteredSkills, customSkill)

        return session.SkillsResult{
            Skills:      filteredSkills,
            Diagnostics: current.Diagnostics,
        }
    },
})

if err := resourceLoader.Reload(); err != nil {
    log.Fatal(err)
}

// 查看加载的技能
skillsResult := resourceLoader.GetSkills()
fmt.Printf("加载了 %d 个技能\n", len(skillsResult.Skills))
```

## 禁用模型调用

某些技能可以设置为只能通过 `/skill:name` 命令显式调用：

```go
hiddenSkill := session.SkillInfo{
    Name:        "internal-tool",
    Description: "内部工具",
    FilePath:    "/path/to/SKILL.md",
    BaseDir:     "/path/to",
    Source:      "project",
    DisableModelInvocation: true, // 禁用模型自动调用
}
```

## 技能格式化

将技能格式化为系统提示词的一部分：

```go
// 获取技能
result := session.LoadSkills(session.LoadSkillsOptions{})

// 格式化为 XML 格式
formatted := session.FormatSkillsForPrompt(result.Skills)
fmt.Println(formatted)

// 输出示例：
// <available_skills>
//   <skill>
//     <name>my-skill</name>
//     <description>技能描述</description>
//     <location>/path/to/SKILL.md</location>
//   </skill>
// </available_skills>
```

## 技能名称规范

技能名称必须符合 Agent Skills 规范：

- 只能包含小写字母、数字和连字符
- 不能以连字符开头或结尾
- 不能包含连续连字符
- 最大长度 64 字符
- 必须与父目录名匹配（除非在 frontmatter 中指定）

## 技能冲突检测

SDK 会自动检测同名技能冲突：

```go
result := session.LoadSkills(session.LoadSkillsOptions{})

for _, diag := range result.Diagnostics {
    if diag.Type == "collision" && diag.Collision != nil {
        fmt.Printf("技能冲突: %s\n", diag.Collision.Name)
        fmt.Printf("  保留: %s\n", diag.Collision.WinnerPath)
        fmt.Printf("  忽略: %s\n", diag.Collision.LoserPath)
    }
}
```

## 忽略文件

技能目录支持 `.gitignore`、`.ignore`、`.fdignore` 文件来排除特定文件：

```gitignore
# 忽略测试文件
test/
*.test.md

# 但保留特定文件
!important.test.md
```

## 完整示例

```go
package main

import (
    "context"
    "fmt"
    "log"
    "strings"

    "github.com/jay-y/pi/pkg/ai-agent-session"
)

func boolPtr(b bool) *bool {
    return &b
}

func main() {
    ctx := context.Background()

    // 加载所有技能
    skillsResult := session.LoadSkills(session.LoadSkillsOptions{
        IncludeDefaults: boolPtr(true),
    })

    fmt.Printf("发现 %d 个技能\n", len(skillsResult.Skills))

    // 显示技能列表
    for _, skill := range skillsResult.Skills {
        invocationStatus := ""
        if skill.DisableModelInvocation {
            invocationStatus = " [手动调用]"
        }
        fmt.Printf("  - %s: %s%s\n", skill.Name, skill.Description, invocationStatus)
    }

    // 显示警告
    if len(skillsResult.Diagnostics) > 0 {
        fmt.Println("\n警告:")
        for _, diag := range skillsResult.Diagnostics {
            fmt.Printf("  [%s] %s\n", diag.Type, diag.Message)
        }
    }

    // 创建资源加载器，过滤技能
    resourceLoader := session.NewDefaultResourceLoader(session.DefaultResourceLoaderOptions{
        SkillsOverride: func(current session.SkillsResult) session.SkillsResult {
            // 只保留名称包含 "code" 的技能
            filtered := []session.SkillInfo{}
            for _, skill := range current.Skills {
                if strings.Contains(skill.Name, "code") {
                    filtered = append(filtered, skill)
                }
            }
            return session.SkillsResult{
                Skills:      filtered,
                Diagnostics: current.Diagnostics,
            }
        },
    })

    if err := resourceLoader.Reload(); err != nil {
        log.Fatal(err)
    }

    // 创建会话
    agentSession, err := session.NewAgentSession(&session.AgentSessionConfig{
        ResourceLoader: resourceLoader,
    })
    if err != nil {
        log.Fatal(err)
    }
    defer agentSession.Close()

    fmt.Println("\n会话已创建，使用过滤后的技能")
}
```
