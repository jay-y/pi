# 完整控制

本文档介绍如何在 Go SDK 中完全自定义所有配置，不使用任何自动发现功能。

## 概述

通过完全自定义配置，你可以：

- 完全控制系统提示词
- 精确指定技能和扩展
- 自定义工具集
- 完全控制会话持久化
- 自定义所有设置

## 自定义认证存储

创建自定义位置的认证存储：

```go
// 创建自定义认证存储
authStorage := session.NewAuthStorageWithPath("/tmp/my-agent/auth.json")

// 设置运行时 API 密钥（不会被持久化）
if apiKey := os.Getenv("MY_ANTHROPIC_KEY"); apiKey != "" {
    authStorage.SetRuntimeAPIKey("anthropic", apiKey)
}
```

## 自定义模型注册表

```go
// 创建模型注册表
modelRegistry := session.NewModelRegistry()

// 查找特定模型
model := modelRegistry.Find("anthropic", "claude-sonnet-4-20250514")
if model == nil {
    log.Fatal("模型未找到")
}
```

## 内存设置管理器

创建内存中的设置管理器，完全控制所有设置：

```go
// 创建内存设置管理器
settingsManager := session.NewInMemorySettingsManager(session.InMemorySettingsOptions{
    Compaction: &session.CompactionSettings{
        Enabled: false, // 禁用压缩
    },
    Retry: &session.RetrySettings{
        Enabled:    true,
        MaxRetries: 2,
    },
    Terminal: &session.TerminalSettings{
        Shell: "/bin/zsh",
    },
})
```

## 完全自定义资源加载器

实现 `ResourceLoader` 接口，完全控制所有资源：

```go
// 自定义资源加载器
type CustomResourceLoader struct{}

func (c *CustomResourceLoader) GetExtensions() session.ExtensionsResult {
    return session.ExtensionsResult{
        Extensions: []session.ExtensionInfo{},
        Errors:     []error{},
        Runtime:    session.NewExtensionRuntime(),
    }
}

func (c *CustomResourceLoader) GetSkills() session.SkillsResult {
    return session.SkillsResult{
        Skills:      []session.SkillInfo{},
        Diagnostics: []session.ResourceDiagnostic{},
    }
}

func (c *CustomResourceLoader) GetPrompts() session.PromptsResult {
    return session.PromptsResult{
        Prompts:     []session.PromptTemplate{},
        Diagnostics: []session.ResourceDiagnostic{},
    }
}

func (c *CustomResourceLoader) GetThemes() session.ThemesResult {
    return session.ThemesResult{
        Themes:      []session.Theme{},
        Diagnostics: []session.ResourceDiagnostic{},
    }
}

func (c *CustomResourceLoader) GetAgentsFiles() session.AgentsFilesResult {
    return session.AgentsFilesResult{
        AgentsFiles: []session.AgentsFile{},
    }
}

func (c *CustomResourceLoader) GetSystemPrompt() string {
    return `你是一个极简助手。
可用工具：read, bash。请保持简洁。`
}

func (c *CustomResourceLoader) GetAppendSystemPrompt() []string {
    return []string{}
}

func (c *CustomResourceLoader) GetPathMetadata() map[string]session.PathMetadata {
    return make(map[string]session.PathMetadata)
}

func (c *CustomResourceLoader) ExtendResources(paths session.ResourceExtensionPaths) {
}

func (c *CustomResourceLoader) Reload() error {
    return nil
}
```

## 自定义工具集

只使用特定的工具：

```go
// 创建自定义工具集
// 注意：当使用自定义 cwd 时，必须使用工厂函数创建工具
cwd := "/path/to/project"

tools := []agent.AgentTool{
    // 使用工厂函数确保路径解析正确
    createReadTool(cwd),
    createBashTool(cwd),
}
```

## 完整示例

```go
package main

import (
    "context"
    "fmt"
    "log"
    "os"

    "github.com/jay-y/pi/pkg/ai"
    agent "github.com/jay-y/pi/pkg/ai-agent"
    "github.com/jay-y/pi/pkg/ai-agent-session"
)

// 自定义资源加载器
type MinimalResourceLoader struct{}

func (m *MinimalResourceLoader) GetExtensions() session.ExtensionsResult {
    return session.ExtensionsResult{
        Extensions: []session.ExtensionInfo{},
        Errors:     []error{},
        Runtime:    session.NewExtensionRuntime(),
    }
}

func (m *MinimalResourceLoader) GetSkills() session.SkillsResult {
    return session.SkillsResult{
        Skills:      []session.SkillInfo{},
        Diagnostics: []session.ResourceDiagnostic{},
    }
}

func (m *MinimalResourceLoader) GetPrompts() session.PromptsResult {
    return session.PromptsResult{
        Prompts:     []session.PromptTemplate{},
        Diagnostics: []session.ResourceDiagnostic{},
    }
}

func (m *MinimalResourceLoader) GetThemes() session.ThemesResult {
    return session.ThemesResult{
        Themes:      []session.Theme{},
        Diagnostics: []session.ResourceDiagnostic{},
    }
}

func (m *MinimalResourceLoader) GetAgentsFiles() session.AgentsFilesResult {
    return session.AgentsFilesResult{
        AgentsFiles: []session.AgentsFile{},
    }
}

func (m *MinimalResourceLoader) GetSystemPrompt() string {
    return `你是一个极简代码助手。

可用工具：
- read: 读取文件内容
- bash: 执行 shell 命令

规则：
1. 保持回答简洁
2. 一次只执行一个工具
3. 等待工具结果后再继续`
}

func (m *MinimalResourceLoader) GetAppendSystemPrompt() []string {
    return []string{}
}

func (m *MinimalResourceLoader) GetPathMetadata() map[string]session.PathMetadata {
    return make(map[string]session.PathMetadata)
}

func (m *MinimalResourceLoader) ExtendResources(paths session.ResourceExtensionPaths) {
}

func (m *MinimalResourceLoader) Reload() error {
    return nil
}

func main() {
    ctx := context.Background()

    // 自定义认证存储
    authStorage := session.NewAuthStorageWithPath("/tmp/my-agent/auth.json")
    if apiKey := os.Getenv("ANTHROPIC_API_KEY"); apiKey != "" {
        authStorage.SetRuntimeAPIKey("anthropic", apiKey)
    }

    // 模型注册表
    modelRegistry := session.NewModelRegistry()
    model := modelRegistry.Find("anthropic", "claude-sonnet-4-20250514")
    if model == nil {
        log.Fatal("模型未找到")
    }

    // 内存设置管理器
    settingsManager := session.NewInMemorySettingsManager(session.InMemorySettingsOptions{
        Compaction: &session.CompactionSettings{
            Enabled: false,
        },
        Retry: &session.RetrySettings{
            Enabled:    true,
            MaxRetries: 2,
        },
    })

    // 工作目录
    cwd := "/path/to/project"

    // 自定义资源加载器
    resourceLoader := &MinimalResourceLoader{}

    // 内存会话管理器
    sessionManager := session.NewInMemorySessionManager()

    // 创建会话
    agentSession, err := session.NewAgentSession(&session.AgentSessionConfig{
        Cwd:             cwd,
        AgentDir:        "/tmp/my-agent",
        Model:           model,
        ThinkingLevel:   session.ThinkingLevelOff,
        AuthStorage:     authStorage,
        ModelRegistry:   modelRegistry,
        ResourceLoader:  resourceLoader,
        Tools:           []agent.AgentTool{}, // 自定义工具
        SessionManager:  sessionManager,
        SettingsManager: settingsManager,
    })
    if err != nil {
        log.Fatal(err)
    }
    defer agentSession.Close()

    // 订阅事件
    agentSession.Subscribe(func(event session.AgentSessionEvent) {
        if event.Type == session.EventTypeMessageUpdate {
            if msgEvent, ok := event.AssistantMessageEvent.(*session.TextDeltaEvent); ok {
                fmt.Print(msgEvent.Delta)
            }
        }
    })

    // 发送提示
    if err := agentSession.Prompt(ctx, "你好！"); err != nil {
        log.Fatal(err)
    }
    fmt.Println()
}
```

## 使用 DefaultResourceLoader 的部分自定义

如果不想完全自定义，可以使用 `DefaultResourceLoader` 的部分覆盖功能：

```go
resourceLoader := session.NewDefaultResourceLoader(session.DefaultResourceLoaderOptions{
    // 完全替换系统提示词
    SystemPromptOverride: func(base string) string {
        return "自定义系统提示词"
    },

    // 覆盖技能
    SkillsOverride: func(current session.SkillsResult) session.SkillsResult {
        return session.SkillsResult{
            Skills: []session.SkillInfo{
                {
                    Name:        "custom-skill",
                    Description: "自定义技能",
                    FilePath:    "/path/to/skill.md",
                    BaseDir:     "/path/to",
                    Source:      "custom",
                },
            },
        }
    },

    // 覆盖上下文文件
    AgentsFilesOverride: func(current session.AgentsFilesResult) session.AgentsFilesResult {
        return session.AgentsFilesResult{
            AgentsFiles: []session.AgentsFile{
                {
                    Path:    "/virtual/AGENTS.md",
                    Content: "# 自定义上下文",
                },
            },
        }
    },

    // 覆盖提示词模板
    PromptsOverride: func(current session.PromptsResult) session.PromptsResult {
        return session.PromptsResult{
            Prompts: []session.PromptTemplate{
                {
                    Name:        "custom-prompt",
                    Description: "自定义提示词",
                    Content:     "提示词内容",
                },
            },
        }
    },
})

if err := resourceLoader.Reload(); err != nil {
    log.Fatal(err)
}
```

## 配置选项总结

| 选项              | 说明         | 默认值         |
| ----------------- | ------------ | -------------- |
| `Cwd`             | 工作目录     | 当前目录       |
| `AgentDir`        | 代理配置目录 | `~/.pi/agent`  |
| `Model`           | 使用的模型   | 从设置中选择   |
| `ThinkingLevel`   | 思考级别     | 从设置中选择   |
| `AuthStorage`     | 认证存储     | 默认位置       |
| `ModelRegistry`   | 模型注册表   | 默认注册表     |
| `ResourceLoader`  | 资源加载器   | 默认加载器     |
| `Tools`           | 工具集       | 完整工具集     |
| `SessionManager`  | 会话管理器   | 新建持久化会话 |
| `SettingsManager` | 设置管理器   | 文件设置管理器 |
