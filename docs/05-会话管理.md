# 会话管理

本文档介绍如何在 Go SDK 中管理会话的持久化和生命周期。

## 会话类型

SDK 支持多种会话管理方式：

1. **内存会话** - 不持久化，程序结束后数据丢失
2. **新建持久化会话** - 创建新的持久化会话文件
3. **继续最近会话** - 继续上次的会话
4. **打开特定会话** - 打开指定的会话文件

## 内存会话

内存会话不会在磁盘上创建文件，适合临时使用：

```go
// 创建内存会话管理器
sessionManager := session.NewInMemorySessionManager()

// 创建会话
agentSession, err := session.NewAgentSession(&session.AgentSessionConfig{
    SessionManager: sessionManager,
})
if err != nil {
    log.Fatal(err)
}

// 内存会话没有会话文件
fmt.Printf("会话ID: %s\n", agentSession.SessionID)
fmt.Printf("会话文件: %s\n", agentSession.SessionFile) // 空字符串
```

## 新建持久化会话

创建新的持久化会话，自动保存到磁盘：

```go
// 创建持久化会话管理器
sessionManager := session.NewFileSessionManager("/path/to/project")

// 创建会话
agentSession, err := session.NewAgentSession(&session.AgentSessionConfig{
    SessionManager: sessionManager,
})
if err != nil {
    log.Fatal(err)
}

// 持久化会话有会话文件
fmt.Printf("会话文件: %s\n", agentSession.SessionFile)
```

### 自定义会话目录

可以指定自定义的会话存储目录：

```go
// 使用自定义会话目录
sessionManager := session.NewFileSessionManagerWithDir("/path/to/project", "/custom/sessions/dir")
```

## 继续最近会话

自动找到并继续最近的会话：

```go
// 继续最近的会话
sessionManager := session.NewContinueRecentSessionManager("/path/to/project")

// 创建会话
agentSession, err := session.NewAgentSession(&session.AgentSessionConfig{
    SessionManager: sessionManager,
})
if err != nil {
    log.Fatal(err)
}

// 检查是否有回退消息
if agentSession.ModelFallbackMessage != "" {
    fmt.Printf("注意: %s\n", agentSession.ModelFallbackMessage)
}

fmt.Printf("继续会话: %s\n", agentSession.SessionFile)
```

如果没有最近的会话，会自动创建新会话。

## 打开特定会话

打开指定的会话文件：

```go
// 打开特定会话
sessionManager := session.NewOpenSessionManager("/path/to/session.jsonl")

// 创建会话
agentSession, err := session.NewAgentSession(&session.AgentSessionConfig{
    SessionManager: sessionManager,
})
if err != nil {
    log.Fatal(err)
}

fmt.Printf("打开会话: %s\n", agentSession.SessionID)
```

## 列出所有会话

查看项目中的所有会话：

```go
// 列出所有会话
sessions, err := session.ListSessions("/path/to/project")
if err != nil {
    log.Fatal(err)
}

fmt.Printf("找到 %d 个会话:\n", len(sessions))
for _, info := range sessions {
    // 截断显示
    firstMsg := info.FirstMessage
    if len(firstMsg) > 30 {
        firstMsg = firstMsg[:30] + "..."
    }
    fmt.Printf("  %s... - \"%s\"\n", info.ID[:8], firstMsg)
}
```

## 会话数据结构

会话使用 JSONL 格式存储，每条记录是一个 JSON 对象：

```jsonl
{"type":"header","version":"v3","id":"abc123","createdAt":"2024-01-01T00:00:00Z"}
{"type":"message","id":"msg1","parentId":null,"role":"user","content":"你好"}
{"type":"message","id":"msg2","parentId":"msg1","role":"assistant","content":"你好！有什么可以帮助你的？"}
```

## 会话压缩

长时间会话可以启用自动压缩来管理上下文窗口：

```go
// 创建设置管理器，启用压缩
settingsManager := session.NewSettingsManager(session.SettingsManagerOptions{
    Compaction: &session.CompactionSettings{
        Enabled: true,
        TriggerTokens: 8000,
        TargetTokens: 4000,
    },
})

// 创建会话
agentSession, err := session.NewAgentSession(&session.AgentSessionConfig{
    SettingsManager: settingsManager,
})
```

## 会话事件

会话管理器会触发各种事件：

```go
agentSession.Subscribe(func(event session.AgentSessionEvent) {
    switch event.Type {
    case session.EventTypeSessionCreated:
        fmt.Println("会话已创建")
    case session.EventTypeMessageAdded:
        fmt.Printf("消息已添加: %s\n", event.MessageID)
    case session.EventTypeCompact:
        fmt.Println("会话已压缩")
    case session.EventTypeBranchCreated:
        fmt.Printf("分支已创建: %s\n", event.BranchID)
    }
})
```

## 分支管理

从特定消息创建分支：

```go
// 从特定消息创建分支
branchID, err := agentSession.CreateBranch("message-id", "分支描述")
if err != nil {
    log.Fatal(err)
}
fmt.Printf("创建分支: %s\n", branchID)

// 切换分支
err = agentSession.SwitchBranch(branchID)
if err != nil {
    log.Fatal(err)
}

// 获取分支列表
branches := agentSession.GetBranches()
for _, branch := range branches {
    fmt.Printf("分支: %s - %s\n", branch.ID, branch.Description)
}
```

## 完整示例

```go
package main

import (
    "context"
    "fmt"
    "log"
    "os"
    "os/signal"
    "syscall"

    "github.com/jay-y/pi/pkg/ai-agent/session"
)

func main() {
    ctx, cancel := context.WithCancel(context.Background())
    defer cancel()

    // 处理中断信号
    sigChan := make(chan os.Signal, 1)
    signal.Notify(sigChan, syscall.SIGINT, syscall.SIGTERM)
    go func() {
        <-sigChan
        cancel()
    }()

    cwd := "/path/to/project"

    // 尝试继续最近的会话
    sessionManager := session.NewContinueRecentSessionManager(cwd)

    agentSession, err := session.NewAgentSession(&session.AgentSessionConfig{
        SessionManager: sessionManager,
    })
    if err != nil {
        log.Fatal(err)
    }
    defer agentSession.Close()

    // 显示会话信息
    if agentSession.SessionFile != "" {
        fmt.Printf("会话文件: %s\n", agentSession.SessionFile)
    } else {
        fmt.Println("内存会话（无持久化）")
    }

    // 订阅事件
    agentSession.Subscribe(func(event session.AgentSessionEvent) {
        if event.Type == session.EventTypeMessageUpdate {
            if msgEvent, ok := event.AssistantMessageEvent.(*session.TextDeltaEvent); ok {
                fmt.Print(msgEvent.Delta)
            }
        }
    })

    // 发送提示
    if err := agentSession.Prompt(ctx, "请列出当前目录的文件"); err != nil {
        log.Fatal(err)
    }
    fmt.Println()

    // 列出所有会话
    sessions, err := session.ListSessions(cwd)
    if err != nil {
        log.Fatal(err)
    }
    fmt.Printf("\n项目共有 %d 个会话\n", len(sessions))
}
```

## 会话清理

删除旧的会话文件：

```go
// 列出所有会话
sessions, err := session.ListSessions("/path/to/project")
if err != nil {
    log.Fatal(err)
}

// 删除除最近3个外的所有会话
if len(sessions) > 3 {
    for _, s := range sessions[3:] {
        if err := os.Remove(s.Path); err != nil {
            log.Printf("删除会话失败 %s: %v", s.Path, err)
        }
    }
}
```
