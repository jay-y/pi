# 模型选择

本文档介绍如何在 Go SDK 中选择和配置 AI 模型。

## 模型配置

在 SDK 中，你需要直接创建和配置模型：

```go
import (
    "github.com/jay-y/pi/pkg/ai"
)

// 创建 Ollama 模型
ollamaModel := &ai.BaseModel{
    ID:            "qwen3-coder-next:q8_0",
    Name:          "ollama/qwen3-coder-next:q8_0",
    API:           ai.ModelApi(ai.ApiOpenAICompletions),
    Provider:      ai.ModelProvider("ollama"),
    BaseURL:       "http://127.0.0.1:11434/v1",
    Reasoning:     false,
    Input:         []string{"text"},
    Cost:          ai.ModelCost{},
    ContextWindow: 128000,
    MaxTokens:     32000,
}

// 创建其他模型（例如 OpenAI）
openaiModel := &ai.BaseModel{
    ID:            "gpt-4o",
    Name:          "openai/gpt-4o",
    API:           ai.ModelApi(ai.ApiOpenAICompletions),
    Provider:      ai.ModelProvider("openai"),
    BaseURL:       "https://api.openai.com/v1",
    APIKey:        "your-api-key",
    Reasoning:     true,
    Input:         []string{"text"},
    Cost:          ai.ModelCost{},
    ContextWindow: 128000,
    MaxTokens:     32000,
}
```

## 获取可用模型

获取所有可用模型（有有效 API 密钥的模型）：

```go
// 获取可用模型
availableModels := modelRegistry.GetAvailable()
fmt.Println("可用模型:")
for _, m := range availableModels {
    fmt.Printf("  - %s/%s\n", m.Provider, m.ID)
}
```

## 选择模型

创建代理和会话时使用模型：

```go
import (
    "github.com/jay-y/pi/pkg/ai"
    agent "github.com/jay-y/pi/pkg/ai-agent"
    session "github.com/jay-y/pi/pkg/ai-agent-session"
)

// 创建模型
ollamaModel := &ai.BaseModel{
    ID:            "qwen3-coder-next:q8_0",
    Name:          "ollama/qwen3-coder-next:q8_0",
    API:           ai.ModelApi(ai.ApiOpenAICompletions),
    Provider:      ai.ModelProvider("ollama"),
    BaseURL:       "http://127.0.0.1:11434/v1",
    Reasoning:     true, // 启用思考
    Input:         []string{"text"},
    Cost:          ai.ModelCost{},
    ContextWindow: 128000,
    MaxTokens:     32000,
    Compat: &session.OpenAICompletionsCompat{
        ThinkingFormat: "qwen", // 思考格式
    },
}

// 创建代理
al := agent.NewAgent(agent.AgentOptions{
    InitialState: &agent.AgentState{
        SystemPrompt: "你是一个助手。一步一步完成任务。",
        Model:        ollamaModel,
    },
})

// 创建会话
cwd, _ := os.Getwd()
agentSession := session.NewAgentSession(&session.AgentSessionConfig{
    Agent:           al,
    ModelRegistry:   &session.ModelRegistry{},
    SessionManager:  session.InMemorySessionManager(cwd),
    SettingsManager: &session.SettingsManager{},
})
```

## 思考级别

思考级别通过模型的 `Reasoning` 字段控制：

```go
// 启用思考
ollamaModel := &ai.BaseModel{
    // 其他配置...
    Reasoning: true,
    Compat: &session.OpenAICompletionsCompat{
        ThinkingFormat: "qwen", // 思考格式，支持 qwen、claude 等
    },
}

// 禁用思考
ollamaModel := &ai.BaseModel{
    // 其他配置...
    Reasoning: false,
}
```

## 自定义模型

可以通过 `models.json` 文件添加自定义模型：

```go
// 模型注册表会自动加载 ~/.pi/agent/models.json 中的自定义模型
// 文件格式示例：
// {
//     "models": [
//         {
//             "provider": "my-provider",
//             "id": "my-model",
//             "name": "My Custom Model",
//             "contextWindow": 128000
//         }
//     ]
// }

// 查找自定义模型
customModel := modelRegistry.Find("my-provider", "my-model")
if customModel != nil {
    fmt.Printf("找到自定义模型: %s\n", customModel.Name)
}
```

## API 密钥管理

模型注册表需要配合认证存储使用：

```go
// 创建认证存储
authStorage := session.NewAuthStorage()

// 设置 API 密钥
authStorage.SetAPIKey("anthropic", "your-api-key")

// 创建模型注册表
modelRegistry := session.NewModelRegistry()

// 现在可以获取可用模型
availableModels := modelRegistry.GetAvailable()
```

## 运行时 API 密钥

可以设置临时的运行时 API 密钥（不会被持久化）：

```go
// 从环境变量获取密钥
apiKey := os.Getenv("ANTHROPIC_API_KEY")
if apiKey != "" {
    authStorage.SetRuntimeAPIKey("anthropic", apiKey)
}
```

## 完整示例

```go
package main

import (
    "context"
    "fmt"
    "log"
    "os"

    "github.com/jay-y/pi/pkg/ai"
    agent "github.com/jay-y/pi/pkg/ai-agent"
    session "github.com/jay-y/pi/pkg/ai-agent-session"
    tools "github.com/jay-y/pi/pkg/ai-agent-tools"
)

func main() {
    ctx := context.Background()

    // 注册内置 API 提供者
    ai.RegisterBuiltInApiProviders()

    // 创建模型
    ollamaModel := &ai.BaseModel{
        ID:            "qwen3-coder-next:q8_0",
        Name:          "ollama/qwen3-coder-next:q8_0",
        API:           ai.ModelApi(ai.ApiOpenAICompletions),
        Provider:      ai.ModelProvider("ollama"),
        BaseURL:       "http://127.0.0.1:11434/v1",
        Reasoning:     true,
        Input:         []string{"text"},
        Cost:          ai.ModelCost{},
        ContextWindow: 128000,
        MaxTokens:     32000,
        Compat: &session.OpenAICompletionsCompat{
            ThinkingFormat: "qwen",
        },
    }

    // 创建代理
    al := agent.NewAgent(agent.AgentOptions{
        InitialState: &agent.AgentState{
            SystemPrompt: "你是一个助手。一步一步完成任务。",
            Model:        ollamaModel,
        },
    })

    // 设置工具
    cwd, err := os.Getwd()
    if err != nil {
        log.Fatal(err)
    }
    al.SetTools(tools.CreateCodingTools(cwd))

    // 创建会话
    agentSession := session.NewAgentSession(&session.AgentSessionConfig{
        Agent:           al,
        ModelRegistry:   &session.ModelRegistry{},
        SessionManager:  session.InMemorySessionManager(cwd),
        SettingsManager: &session.SettingsManager{},
    })
    defer agentSession.Dispose()

    // 订阅事件
    agentSession.Subscribe(func(event session.AgentSessionEvent) {
        switch ae := event.(type) {
        case *agent.AgentEventMessageUpdate:
            switch e := ae.AssistantMessageEvent.(type) {
            case *ai.AssistantMessageEventTextDelta:
                fmt.Print(e.Delta)
            }
        }
    })

    // 发送提示
    if err := agentSession.Prompt(ctx, "你好！", nil); err != nil {
        log.Fatal(err)
    }
    fmt.Println()
}
```

## 模型配置注意事项

1. **API 密钥管理**：对于需要 API 密钥的模型（如 OpenAI），直接在模型配置中设置 `APIKey` 字段

2. **思考格式**：当启用思考时，需要设置 `ThinkingFormat` 以匹配模型的输出格式

3. **BaseURL**：对于本地模型（如 Ollama），需要设置正确的本地 URL

4. **上下文窗口**：根据模型的实际能力设置 `ContextWindow` 和 `MaxTokens`

5. **API 提供者**：确保注册了对应的 API 提供者，对于内置提供者使用 `ai.RegisterBuiltInApiProviders()`
