# 模型选择

本文档介绍如何在 Go SDK 中选择和配置 AI 模型。

## 模型注册表

SDK 使用 `ModelRegistry` 来管理内置模型和自定义模型：

```go
// 创建模型注册表
modelRegistry := session.NewModelRegistry()

// 查找特定模型
model := modelRegistry.Find("anthropic", "claude-opus-4-5")
if model != nil {
    fmt.Printf("找到模型: %s/%s\n", model.Provider, model.ID)
}
```

## 获取可用模型

获取所有可用模型（有有效 API 密钥的模型）：

```go
// 获取可用模型
availableModels := modelRegistry.GetAvailable()
fmt.Println("可用模型:")
for _, m := range availableModels {
    fmt.Printf("  - %s/%s\n", m.Provider, m.ID)
}
```

## 选择模型

创建会话时指定模型：

```go
// 创建模型注册表
modelRegistry := session.NewModelRegistry()

// 查找特定模型
model := modelRegistry.Find("anthropic", "claude-opus-4-5")
if model == nil {
    log.Fatal("模型未找到")
}

// 创建会话，指定模型和思考级别
agentSession, err := session.NewAgentSession(&session.AgentSessionConfig{
    Model:         model,
    ThinkingLevel: session.ThinkingLevelMedium, // off, low, medium, high
    ModelRegistry: modelRegistry,
})
```

## 思考级别

思考级别控制模型在响应前的推理深度：

```go
const (
    ThinkingLevelOff    ThinkingLevel = "off"    // 不思考
    ThinkingLevelLow    ThinkingLevel = "low"    // 少量思考
    ThinkingLevelMedium ThinkingLevel = "medium" // 中等思考
    ThinkingLevelHigh   ThinkingLevel = "high"   // 深度思考
)
```

## 自定义模型

可以通过 `models.json` 文件添加自定义模型：

```go
// 模型注册表会自动加载 ~/.pi/agent/models.json 中的自定义模型
// 文件格式示例：
// {
//     "models": [
//         {
//             "provider": "my-provider",
//             "id": "my-model",
//             "name": "My Custom Model",
//             "contextWindow": 128000
//         }
//     ]
// }

// 查找自定义模型
customModel := modelRegistry.Find("my-provider", "my-model")
if customModel != nil {
    fmt.Printf("找到自定义模型: %s\n", customModel.Name)
}
```

## API 密钥管理

模型注册表需要配合认证存储使用：

```go
// 创建认证存储
authStorage := session.NewAuthStorage()

// 设置 API 密钥
authStorage.SetAPIKey("anthropic", "your-api-key")

// 创建模型注册表
modelRegistry := session.NewModelRegistry()

// 现在可以获取可用模型
availableModels := modelRegistry.GetAvailable()
```

## 运行时 API 密钥

可以设置临时的运行时 API 密钥（不会被持久化）：

```go
// 从环境变量获取密钥
apiKey := os.Getenv("ANTHROPIC_API_KEY")
if apiKey != "" {
    authStorage.SetRuntimeAPIKey("anthropic", apiKey)
}
```

## 完整示例

```go
package main

import (
    "context"
    "fmt"
    "log"
    "os"

    "github.com/jay-y/pi/pkg/ai-agent-session"
)

func main() {
    ctx := context.Background()

    // 创建认证存储
    authStorage := session.NewAuthStorage()

    // 从环境变量设置 API 密钥
    if apiKey := os.Getenv("ANTHROPIC_API_KEY"); apiKey != "" {
        authStorage.SetRuntimeAPIKey("anthropic", apiKey)
    }

    // 创建模型注册表
    modelRegistry := session.NewModelRegistry()

    // 获取可用模型
    availableModels := modelRegistry.GetAvailable()
    if len(availableModels) == 0 {
        log.Fatal("没有可用的模型，请配置 API 密钥")
    }

    fmt.Println("可用模型:")
    for _, m := range availableModels {
        fmt.Printf("  - %s/%s (%s)\n", m.Provider, m.ID, m.Name)
    }

    // 使用第一个可用模型
    model := availableModels[0]
    fmt.Printf("\n使用模型: %s/%s\n", model.Provider, model.ID)

    // 创建会话
    agentSession, err := session.NewAgentSession(&session.AgentSessionConfig{
        Model:         model,
        ThinkingLevel: session.ThinkingLevelMedium,
        ModelRegistry: modelRegistry,
    })
    if err != nil {
        log.Fatal(err)
    }
    defer agentSession.Close()

    // 订阅事件
    agentSession.Subscribe(func(event session.AgentSessionEvent) {
        if event.Type == session.EventTypeMessageUpdate {
            if msgEvent, ok := event.AssistantMessageEvent.(*session.TextDeltaEvent); ok {
                fmt.Print(msgEvent.Delta)
            }
        }
    })

    // 发送提示
    if err := agentSession.Prompt(ctx, "你好！"); err != nil {
        log.Fatal(err)
    }
    fmt.Println()
}
```

## 模型回退

如果指定的模型不可用，SDK 会自动回退到其他可用模型：

```go
agentSession, err := session.NewAgentSession(&session.AgentSessionConfig{
    Model:         model,
    ModelRegistry: modelRegistry,
})
if err != nil {
    log.Fatal(err)
}

// 检查是否有回退消息
if agentSession.ModelFallbackMessage != "" {
    fmt.Printf("注意: %s\n", agentSession.ModelFallbackMessage)
}
```
