# 系统提示词

本文档介绍如何在 Go SDK 中自定义系统提示词。

## 默认系统提示词

SDK 会自动从以下位置加载系统提示词：

- `~/.pi/agent/SYSTEM.md` - 全局系统提示词
- `<cwd>/.pi/SYSTEM.md` - 项目级系统提示词
- `~/.pi/agent/APPEND_SYSTEM.md` - 全局追加提示词
- `<cwd>/.pi/APPEND_SYSTEM.md` - 项目级追加提示词

## 完全替换系统提示词

使用 `DefaultResourceLoader` 可以完全替换系统提示词：

```go
// 创建资源加载器，完全替换系统提示词
resourceLoader := session.NewDefaultResourceLoader(session.DefaultResourceLoaderOptions{
    SystemPromptOverride: func(base string) string {
        return `你是一个像海盗一样说话的助手。
总是以 "Arrr!" 结束回答。`
    },
    // 禁用追加提示词
    AppendSystemPromptOverride: func(base []string) []string {
        return []string{}
    },
})

// 重新加载资源
if err := resourceLoader.Reload(); err != nil {
    log.Fatal(err)
}

// 创建会话
agentSession, err := session.NewAgentSession(&session.AgentSessionConfig{
    ResourceLoader: resourceLoader,
})
```

## 追加到默认提示词

在默认提示词基础上添加自定义指令：

```go
resourceLoader := session.NewDefaultResourceLoader(session.DefaultResourceLoaderOptions{
    AppendSystemPromptOverride: func(base []string) []string {
        additionalInstructions := []string{
            "## 额外指令",
            "- 总是简洁回答",
            "- 使用项目符号列出事项",
        }
        return append(base, additionalInstructions...)
    },
})

if err := resourceLoader.Reload(); err != nil {
    log.Fatal(err)
}

agentSession, err := session.NewAgentSession(&session.AgentSessionConfig{
    ResourceLoader: resourceLoader,
})
```

## 获取当前系统提示词

可以查看当前加载的系统提示词：

```go
// 获取系统提示词
systemPrompt := resourceLoader.GetSystemPrompt()
fmt.Printf("系统提示词长度: %d 字符\n", len(systemPrompt))

// 获取追加提示词
appendPrompts := resourceLoader.GetAppendSystemPrompt()
fmt.Printf("追加提示词数量: %d\n", len(appendPrompts))
```

## 上下文文件 (AGENTS.md)

上下文文件提供项目特定的指令，会被自动加载到系统提示词中：

```go
// 自定义上下文文件
resourceLoader := session.NewDefaultResourceLoader(session.DefaultResourceLoaderOptions{
    AgentsFilesOverride: func(current session.AgentsFilesResult) session.AgentsFilesResult {
        // 添加虚拟上下文文件
        virtualFile := session.AgentsFile{
            Path: "/virtual/AGENTS.md",
            Content: `# 项目规范

## 代码风格
- 使用 TypeScript 严格模式
- 不使用 any 类型
- 优先使用 const 而不是 let`,
        }

        return session.AgentsFilesResult{
            AgentsFiles: append(current.AgentsFiles, virtualFile),
        }
    },
})

if err := resourceLoader.Reload(); err != nil {
    log.Fatal(err)
}

// 查看发现的上下文文件
discovered := resourceLoader.GetAgentsFiles()
fmt.Println("发现的上下文文件:")
for _, file := range discovered.AgentsFiles {
    fmt.Printf("  - %s (%d 字符)\n", file.Path, len(file.Content))
}
```

## 禁用上下文文件

完全禁用上下文文件发现：

```go
resourceLoader := session.NewDefaultResourceLoader(session.DefaultResourceLoaderOptions{
    AgentsFilesOverride: func(current session.AgentsFilesResult) session.AgentsFilesResult {
        return session.AgentsFilesResult{
            AgentsFiles: []session.AgentsFile{},
        }
    },
})
```

## 完整示例

```go
package main

import (
    "context"
    "fmt"
    "log"

    "github.com/jay-y/pi/pkg/ai-agent/session"
)

func main() {
    ctx := context.Background()

    // 创建自定义资源加载器
    resourceLoader := session.NewDefaultResourceLoader(session.DefaultResourceLoaderOptions{
        SystemPromptOverride: func(base string) string {
            return `你是一个专业的代码审查助手。

你的职责：
1. 分析代码质量和潜在问题
2. 提供改进建议
3. 遵循最佳实践

回答格式：
- 问题：描述发现的问题
- 建议：提供改进方案
- 示例：给出代码示例`
        },
        AppendSystemPromptOverride: func(base []string) []string {
            return []string{
                "## 额外要求",
                "- 使用中文回答",
                "- 保持专业和礼貌",
            }
        },
    })

    if err := resourceLoader.Reload(); err != nil {
        log.Fatal(err)
    }

    // 查看系统提示词
    systemPrompt := resourceLoader.GetSystemPrompt()
    fmt.Printf("系统提示词:\n%s\n\n", systemPrompt)

    // 创建会话
    agentSession, err := session.NewAgentSession(&session.AgentSessionConfig{
        ResourceLoader: resourceLoader,
    })
    if err != nil {
        log.Fatal(err)
    }
    defer agentSession.Close()

    // 订阅事件
    agentSession.Subscribe(func(event session.AgentSessionEvent) {
        if event.Type == session.EventTypeMessageUpdate {
            if msgEvent, ok := event.AssistantMessageEvent.(*session.TextDeltaEvent); ok {
                fmt.Print(msgEvent.Delta)
            }
        }
    })

    // 发送提示
    if err := agentSession.Prompt(ctx, "请审查以下代码..."); err != nil {
        log.Fatal(err)
    }
    fmt.Println()
}
```

## 提示词模板

SDK 还支持提示词模板，可以从文件加载预定义的提示词：

```go
// 获取所有提示词模板
promptsResult := resourceLoader.GetPrompts()
for _, prompt := range promptsResult.Prompts {
    fmt.Printf("提示词: %s - %s\n", prompt.Name, prompt.Description)
}
```
