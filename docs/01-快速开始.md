# 快速开始

本文档介绍如何使用 Go SDK 创建和运行 AI 代理会话。

## 最小化使用

最简单的使用方式，使用所有默认配置：

```go
package main

import (
	"context"
	"fmt"
	"log"

	"github.com/jay-y/pi/pkg/ai-agent-session"
)

func main() {
	// 创建默认会话
	agentSession, err := session.NewAgentSession(&session.AgentSessionConfig{})
	if err != nil {
		log.Fatal(err)
	}

	// 订阅事件
	agentSession.Subscribe(func(event session.AgentSessionEvent) {
		if event.Type == session.EventTypeMessageUpdate {
			if msgEvent, ok := event.AssistantMessageEvent.(*session.TextDeltaEvent); ok {
				fmt.Print(msgEvent.Delta)
			}
		}
	})

	// 发送提示
	ctx := context.Background()
	if err := agentSession.Prompt(ctx, "当前目录有哪些文件？"); err != nil {
		log.Fatal(err)
	}

	// 查看消息历史
	for _, msg := range agentSession.State.Messages {
		fmt.Printf("%+v\n", msg)
	}
}
```

## 默认行为

使用默认配置时，SDK 会自动：

1. **发现技能**：从当前工作目录的 `.pi/skills` 和 `~/.pi/agent/skills` 加载技能
2. **加载扩展**：从 `.pi/extensions` 和 `~/.pi/agent/extensions` 加载扩展
3. **读取上下文文件**：自动发现 `AGENTS.md` 文件
4. **选择模型**：从设置中选择默认模型，或使用第一个可用模型
5. **使用完整工具集**：包含读、写、编辑、bash、grep 等工具

## 会话状态

会话创建后会返回一个 `AgentSession` 实例，包含以下重要属性：

```go
// 会话ID
sessionID := agentSession.SessionID

// 会话文件路径（如果是持久化会话）
sessionFile := agentSession.SessionFile

// 当前状态
state := agentSession.State

// 消息历史
messages := state.Messages
```

## 事件订阅

通过 `Subscribe` 方法可以监听会话事件：

```go
agentSession.Subscribe(func(event session.AgentSessionEvent) {
	switch event.Type {
	case session.EventTypeMessageUpdate:
		// 消息更新事件
		if msgEvent, ok := event.AssistantMessageEvent.(*session.TextDeltaEvent); ok {
			fmt.Print(msgEvent.Delta)
		}
	case session.EventTypeToolStart:
		// 工具开始执行
		fmt.Printf("工具开始: %s\n", event.ToolName)
	case session.EventTypeToolComplete:
		// 工具执行完成
		fmt.Printf("工具完成: %s\n", event.ToolName)
	}
})
```

## 发送提示

使用 `Prompt` 方法向代理发送提示：

```go
ctx := context.Background()
err := agentSession.Prompt(ctx, "请帮我分析这个项目的代码结构")
```

## 清理资源

会话使用完毕后，应该清理资源：

```go
// 关闭会话
agentSession.Close()
```

## 完整示例

```go
package main

import (
	"context"
	"fmt"
	"log"
	"os"
	"os/signal"
	"syscall"

	"github.com/jay-y/pi/pkg/ai-agent-session"
)

func main() {
	// 创建上下文
	ctx, cancel := context.WithCancel(context.Background())
	defer cancel()

	// 处理中断信号
	sigChan := make(chan os.Signal, 1)
	signal.Notify(sigChan, syscall.SIGINT, syscall.SIGTERM)
	go func() {
		<-sigChan
		cancel()
	}()

	// 创建会话
	agentSession, err := session.NewAgentSession(&session.AgentSessionConfig{})
	if err != nil {
		log.Fatal(err)
	}
	defer agentSession.Close()

	// 订阅事件
	agentSession.Subscribe(func(event session.AgentSessionEvent) {
		if event.Type == session.EventTypeMessageUpdate {
			if msgEvent, ok := event.AssistantMessageEvent.(*session.TextDeltaEvent); ok {
				fmt.Print(msgEvent.Delta)
			}
		}
	})

	// 发送提示
	if err := agentSession.Prompt(ctx, "你好，请介绍一下你自己"); err != nil {
		log.Fatal(err)
	}

	fmt.Println("\n完成！")
}
```
